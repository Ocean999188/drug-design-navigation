## 🚀 快速开始

### 选择1: GitHub Pages（推荐初学者）
**优点**: 完全免费、简单易用、与GitHub集成
**网址格式**: `https://[你的用户名].github.io/drug-design-navigation/`

#### 部署步骤:
1. **创建GitHub仓库**: `drug-design-navigation`
2. **运行部署脚本**: 双击 `deploy.bat`
3. **启用GitHub Pages**: Settings → Pages → main分支
4. **等待部署**: 1-2分钟
5. **访问网站**: 获得永久免费链接

### 选择2: Vercel（推荐专业使用）
**优点**: 全球CDN、自动HTTPS、极速访问
**网址格式**: `https://drug-design-navigation.vercel.app`

#### 部署步骤:
1. **确保GitHub仓库已存在**
2. **登录Vercel**: https://vercel.com
3. **导入项目**: 选择GitHub仓库
4. **一键部署**: 等待30秒
5. **获得链接**: 自动生成vercel.app域名

### 选择3: 双平台部署（最佳方案）
同时部署到GitHub Pages和Vercel，获得两个访问链接，互为备份。

## ⚠️ 常见问题解决

### 问题: GitHub Pages 显示404
**解决**:
1. 等待2-3分钟
2. 检查Settings → Pages配置
3. 确保仓库是Public
4. 检查文件是否在根目录

### 问题: 样式不加载
**解决**:
1. 检查控制台错误
2. 确保CSS文件路径正确
3. 清除浏览器缓存
4. 检查文件大小写

### 问题: 外部链接失效
**解决**:
1. 检查目标网站是否可访问
2. 确保链接使用HTTPS
3. 更新为最新链接
