# 药物设计导航平台访问指南

## 🚀 当前访问方式

### 1. 本地网络访问（已启用）
- **服务器地址**: http://192.168.1.6:8000
- **状态**: ✅ 正在运行
- **访问方法**: 在同一局域网内的设备上，打开浏览器输入上述地址

### 2. 如何让别人访问

#### 方法A：局域网内同事/同学
1. 确保对方设备连接到**同一个WiFi/网络**
2. 让对方在浏览器中输入：`http://192.168.1.6:8000`
3. 即可访问药物设计导航平台

#### 方法B：互联网访问（需要配置）
1. **端口转发**（路由器设置）：
   - 登录路由器管理界面（通常是 192.168.1.1）
   - 找到端口转发或虚拟服务器设置
   - 添加规则：外部端口 8000 → 内部 IP 192.168.1.6:8000
   - 然后别人可以通过你的公网IP访问

2. **使用 ngrok**（推荐）：
   ```bash
   # 下载 ngrok：https://ngrok.com/download
   # 运行：
   ngrok http 8000
   ```
   - 会获得一个类似 `https://abc123.ngrok.io` 的链接
   - 分享这个链接给任何人

## 🌐 高级部署选项

### 1. GitHub Pages（免费）
1. 创建 GitHub 仓库
2. 上传所有文件
3. Settings → Pages → 选择 main 分支
4. 获得：`https://用户名.github.io/仓库名`

### 2. Vercel（推荐，免费）
1. 访问 https://vercel.com
2. 导入 GitHub 仓库
3. 自动部署，获得 `项目名.vercel.app`

### 3. Netlify
1. 访问 https://netlify.com
2. 拖拽文件夹上传
3. 获得 `随机名称.netlify.app`

## 📱 访问测试
1. 在手机上打开浏览器
2. 输入：`http://192.168.1.6:8000`
3. 应该能看到响应式设计的药物设计平台

## ⚠️ 注意事项
1. 确保防火墙允许端口 8000
2. 服务器运行时不要关闭命令行窗口
3. 如需长期运行，考虑部署到云平台
4. 局域网访问需要所有设备在同一网络

## 📞 技术支持
- 如果无法访问，请检查：
  - 服务器是否正在运行
  - IP 地址是否正确
  - 防火墙设置
  - 网络连接状态
